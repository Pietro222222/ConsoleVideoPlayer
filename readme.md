# how it works

```
video.mp4
   |
   | -> Convert To Gif -> Play through Viu
   |
   | -> Convert to mp4 -> Play audio

```

cache: $HOME/.console_player/cache/{video-hash}/
                                    -> video.mp3
                                    -> video.gif
# How to use

./consoleplayer video.mp4
